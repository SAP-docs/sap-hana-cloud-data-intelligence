# LEAST

LEAST function returns the least value among the arguments \(`n1`, `n2`, …\).



<a name="loio577b90cb89d4437988e7b1f55d40ab3d__section_q2p_fb4_w4b"/>

## Syntax

`LEAST(n1 [{, n2}…])`



<a name="loio577b90cb89d4437988e7b1f55d40ab3d__section_r2p_fb4_w4b"/>

## Description

Returns the least value among the arguments \(`n1`, `n2`, …\).



<a name="loio577b90cb89d4437988e7b1f55d40ab3d__section_kpr_hb4_w4b"/>

## Example

`LEAST(‘aa’, ‘ab’, ‘ba’, ‘bb’)`

This example returns `aa` as the least value.
