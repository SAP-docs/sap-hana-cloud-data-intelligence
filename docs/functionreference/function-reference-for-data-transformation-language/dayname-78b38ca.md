# DAYNAME

DAYNAME function \(Date & Time\) returns the weekday in English for date `d`.



<a name="loio78b38ca71ae84e18bca27d80b9f36431__section_zyx_lnh_bpb"/>

## Syntax

`DAYNAME(d)`



<a name="loio78b38ca71ae84e18bca27d80b9f36431__section_ozx_lnh_bpb"/>

## Description

Returns the weekday in English for date `d`.



<a name="loio78b38ca71ae84e18bca27d80b9f36431__section_clg_nnh_bpb"/>

## Example

`DAYNAME(TO_DATE(‘2011-05-30’, ‘yyyy-MM-dd’))`

This example returns `Monday` as the weekday for the specified date.
