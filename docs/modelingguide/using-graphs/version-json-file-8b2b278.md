# version.json File

The `version.json` file contains version information.

For example:

```
{
    "version": "2.3.30-dev-0828",
    "buildTime": "2018-08-28T17:10:27",
    "gitCommit": "a1897fcf38788b55be1ce177e32bb2ebc733b28c",
    "platform": "linux"
}
```
